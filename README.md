
# 📝 Flask Blog App

A full-stack blog application built using Flask with AWS cloud services integration for storage, database, and deployment. The project demonstrates real-world cloud architecture using EC2, S3, and RDS.

---

## ✨ Features

- Create, read, update, delete blog posts (CRUD operations)  
- Image upload functionality  
- AWS S3 integration for media storage  
- AWS RDS MySQL database integration  
- Flask backend deployed on AWS EC2  
- Secure IAM-based AWS service access  

---

## 🛠️ Tech Stack

- Python (Flask)  
- MySQL (AWS RDS)  
- AWS S3 (Storage)  
- AWS EC2 (Deployment)  
- HTML, CSS, Bootstrap  

---

## 📦 Installation

```bash
git clone https://github.com/innovateabhi/flask-blog-app.git
cd flask-blog-app
pip install -r requirements.txt
```


## ▶️ Run Locally

Run the project locally

```bash
  python app.py
```




## ☁️ AWS Architecture
- EC2 → Flask backend server
- RDS → MySQL database
- S3 → Image storage
## 🚀 Deployment

- Backend deployed on AWS EC2
- Database hosted on AWS RDS
- Media files stored in AWS S3
- Connected using IAM roles and credentials


## 🔐 AWS Services Used
- EC2 → Application hosting
- RDS → Database management
- S3 → File storage
- IAM → Access control & security
